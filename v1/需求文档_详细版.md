# 文字版升级打怪游戏需求文档（详细版）

## 一、项目概述

### 1.1 项目名称
梦幻文字版 - 回合制RPG文字游戏

### 1.2 项目背景
基于《梦幻西游》经典玩法，开发一款纯文字界面的回合制角色扮演游戏。使用C语言开发，文本文件作为数据存储介质，提供完整的游戏体验。

### 1.3 游戏定位
- **游戏类型**：单机回合制RPG
- **目标用户**：怀旧游戏玩家、文字游戏爱好者、C语言学习者
- **平台**：Windows控制台程序
- **核心玩法**：角色养成、打怪升级、装备收集、任务探索

### 1.4 技术栈
- **语言**：C语言（标准C99）
- **数据存储**：纯文本文件（.txt格式）
- **界面**：Windows控制台（命令行界面）
- **编译器**：GCC（MinGW）

| 属性 | 英文标识 | 作用 |

| 体质 | CON | 影响气血上限和气血恢复速度 |
| 力量 | STR | 影响物理攻击力 |
| 耐力 | DEF | 影响物理防御力 |
| 敏捷 | AGI | 影响出手速度和闪避率 |
| 魔法 | INT | 影响法力上限和法术伤害 |

#### 2.3.2 战斗属性
| 属性 | 作用 |
|------|------|
| 气血 | 当前生命值，归零则死亡 |
| 法力 | 当前魔法值，释放技能消耗 |
| 攻击 | 物理攻击力 |
| 防御 | 物理防御力 |
| 速度 | 决定战斗出手顺序 |
| 法伤 | 法术伤害力 |
| 法防 | 法术防御力 |
| 闪避 | 躲避物理攻击的概率 |
| 暴击 | 造成双倍伤害的概率 |

#### 2.3.3 属性计算公式
```
气血上限 = 基础气血 + 体质 × 5 + 装备加成
法力上限 = 基础法力 + 魔法 × 3 + 装备加成
攻击 = 基础攻击 + 力量 × 0.8 + 装备加成
防御 = 基础防御 + 耐力 × 0.6 + 装备加成
速度 = 基础速度 + 敏捷 × 0.7 + 装备加成
法伤 = 基础法伤 + 魔法 × 0.7 + 装备加成
法防 = 基础法防 + 耐力 × 0.3 + 装备加成
闪避 = 敏捷 × 0.5%
暴击 = 基础暴击率 + 装备加成
```

2. 选择种族（人族/仙族/魔族）
3. 选择门派（根据种族显示可选门派）
4. 确认创建，进入游戏

## 三、战斗系统

### 3.1 战斗规则

- **回合制**：双方轮流行动，每回合可选择一个操作
- **速度判定**：速度高者优先出手，速度相同时随机判定
- **先手优势**：先手方第一回合攻击时伤害提升10%
- **逃跑机制**：成功率 = 自身速度 / (自身速度 + 敌方速度) × 70%，失败则损失行动机会
- **死亡处理**：战斗中死亡损失当前经验的10%，返回城镇复活，气血恢复50%

### 3.2 战斗操作

| 操作 | 效果 | 消耗 |
|------|------|------|
| 普通攻击 | 物理攻击敌方单体 | 0 |
| 技能攻击 | 使用门派技能攻击 | 消耗对应法力 |
| 防御 | 本回合受到伤害减半 | 0 |
| 使用物品 | 使用背包中的药品或道具 | 消耗物品 |
| 召唤召唤兽 | 召唤召唤兽参战 | 0 |
| 召回召唤兽 | 召回召唤兽 | 0 |
| 逃跑 | 尝试逃离战斗 | 0 |

### 3.3 伤害计算公式

```
物理伤害 = (攻击 - 敌方防御 × 0.5) × 随机系数(0.8 ~ 1.2) × 技能系数

法术伤害 = 法伤 × 技能系数 × (1 - 敌方法防 / (法防 + 100)) × 随机系数(0.9 ~ 1.1)

固定伤害 = 技能基础伤害 + 等级 × 技能等级系数

暴击伤害 = 普通伤害 × 1.5
```

### 3.4 战斗流程

```
战斗开始
    ↓
显示双方状态（气血、法力、等级）
    ↓
[玩家回合]
├── 选择操作
├── 执行动作
├── 计算伤害
└── 显示战斗日志
    ↓
[召唤兽回合]（如有）
    ↓
[敌方回合]
├── 敌方AI选择行动
├── 执行动作
└── 计算伤害
    ↓
{胜负判定}
    ↓[胜利]                    ↓[失败]
显示奖励（经验、金币、掉落）  显示失败提示
    ↓                          ↓
获得奖励                      损失经验，返回城镇
    ↓
继续游戏
```

## 七、升级系统

### 7.1 经验获取途径

| 途径 | 经验值 | 说明 |
|------|--------|------|
| 击杀怪物 | 根据怪物等级 | 基础经验来源 |
| 完成任务 | 根据任务类型 | 稳定经验来源 |
| 使用经验丹 | 固定经验值 | 快速升级道具 |
| 日常活动 | 根据活动类型 | 额外经验来源 |

### 7.2 升级所需经验

```
升级经验 = 当前等级 × 当前等级 × 100 + 当前等级 × 500
```

### 7.3 升级效果

- 等级+1
- 基础属性提升（根据种族）
- 获得5点属性点（可自由分配）
- 获得1点技能点（用于学习或升级技能）
- 解锁新技能（特定等级）
- 解锁新地图（特定等级）
- 可穿戴更高级装备

## 十一、游戏流程设计

### 11.1 游戏主流程

```
开始游戏
    ↓
[主菜单]
├── 新游戏 → 创建角色 → 进入游戏
├── 继续游戏 → 读取存档 → 进入游戏
└── 退出游戏
    ↓
游戏主界面
    ↓
[菜单选项]
├── 查看角色属性
├── 查看背包
├── 查看召唤兽
├── 学习/升级技能
├── 商店交易
├── 任务管理
├── 地图移动
├── 查看装备
└── 保存退出
    ↓
地图移动
    ↓
{遇敌判定}
    ↓[是]            ↓[否]
战斗界面        到达目的地
    ↓                ↓
战斗过程        探索地图/对话/接任务
    ↓                ↓
战斗结束        获得任务/物品
    ↓
获得奖励
    ↓
继续游戏
```

### 11.2 战斗流程

```
战斗开始
    ↓
显示战斗界面（双方状态、行动顺序）
    ↓
[玩家回合]
├── 选择操作
│   ├── 普通攻击 → 选择目标 → 执行攻击
│   ├── 技能攻击 → 选择技能 → 选择目标 → 执行技能
│   ├── 防御 → 执行防御
│   ├── 使用物品 → 选择物品 → 选择目标 → 使用物品
│   ├── 召唤召唤兽 → 选择召唤兽 → 召唤
│   ├── 召回召唤兽 → 召回
│   └── 逃跑 → 判定是否成功
    ↓
显示战斗日志
    ↓
[召唤兽回合]（如有）
├── 召唤兽AI选择行动
└── 执行行动
    ↓
[敌方回合]
├── 敌方AI选择行动
└── 执行行动
    ↓
{胜负判定}
    ↓[胜利]                    ↓[失败]
显示胜利界面                  显示失败界面
├── 获得经验                  ├── 损失10%经验
├── 获得金币                  └── 返回城镇
├── 获得掉落物品
└── 升级提示（如达到升级条件）
    ↓
继续游戏
```

## 十四、非功能需求

- 游戏启动时间 ≤ 2秒

- 支持Windows 7及以上版本
- 支持主流分辨率的控制台窗口

- 代码结构清晰，模块划分明确
- 变量命名规范，注释完整
- 数据与逻辑分离，便于扩展

- 存档文件格式验证，防止篡改

- 错误处理完善，避免程序崩溃
