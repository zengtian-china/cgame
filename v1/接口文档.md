- **原型**：`User * read_save_main(void)`
- **形参**：无

- **返回值**：`User *` — 创建成功返回角色信息指针，角色已存在或创建失败返回 NULL

#### 3. `delUser(User *user)`

- **原型**：`void delUser(User *user)`
- **形参**：
  - `user`：`User *` — 角色信息指针
- **返回值**：无
- **功能**：释放角色信息占用的内存

#### 4. `showUser(User *user)`

- **原型**：`void showUser(User *user)`
- **形参**：
  - `user`：`User *` — 角色信息指针
- **返回值**：无
- **功能**：在控制台打印角色的所有属性信息

#### 5. `insertUser(User *user)`

- **原型**：`void insertUser(User *user)`
- **形参**：
  - `user`：`User *` — 角色信息指针
- **返回值**：无
- **功能**：将角色信息写入存档文件（路径：`./save/{角色名}.txt`），包含基础属性、战斗属性和背包数据

## 二、player 模块（角色系统）

### 文件位置
- 头文件：[player.h](file:///c:/Users/zengt/Desktop/game_copy/v1/player.h)
- 实现文件：[player.c](file:///c:/Users/zengt/Desktop/game_copy/v1/player.c)

### 接口函数

#### 1. `levelUp(User *user)`

- **原型**：`void levelUp(User *user)`
- **形参**：
  - `user`：`User *` — 角色信息指针
- **返回值**：无
- **功能**：角色升级处理，根据当前等级范围增加属性点、金币、HP/MP上限、攻击力、防御力、速度，然后调用 `stat_growth` 让玩家分配额外属性点

#### 2. `levelUpLogic(User *user, int exp)`

- **原型**：`void levelUpLogic(User *user, int exp)`
- **形参**：
  - `user`：`User *` — 角色信息指针
  - `exp`：`int` — 获得的经验值
- **返回值**：无
- **功能**：升级逻辑处理，将获得的经验值累加到角色经验，判断是否达到升级条件，若满足则循环调用 `levelUp` 进行升级

#### 3. `stat_growth(User *user)`

- **原型**：`void stat_growth(User *user)`
- **形参**：
  - `user`：`User *` — 角色信息指针
- **返回值**：无
- **功能**：属性成长分配，让玩家自由分配 5 点属性点到力量、体质、敏捷，然后根据新属性重新计算攻击力、HP上限、速度

#### 4. `UserInit(const char *name)`

- **原型**：`User * UserInit(const char *name)`
- **形参**：
  - `name`：`const char *` — 角色名称
- **返回值**：`User *` — 创建成功返回角色指针，失败返回 NULL
- **功能**：初始化新角色的基础属性（等级1、经验0、金币1000、初始属性点等），然后调用 `insertUser` 保存到文件

- **形参**：

### 接口函数

#### 1. `MonsterInit(const char *filename)`

- **原型**：`int MonsterInit(const char *filename)`
- **形参**：
  - `filename`：`const char *` — 怪物数据文件路径
- **返回值**：`int` — 成功返回加载的怪物数量，失败返回 -1
- **功能**：从指定文件加载所有怪物数据到全局数组 `g_monster`

  - `rates`：`int[]` — 对应怪物的出现概率（总和应为 100）

- **返回值**：`monster *` — 抽中的怪物指针，失败返回 NULL 或越界指针（存在 bug）
- **功能**：根据概率随机从怪物池中抽取一只怪物

#### 3. `GetMonsterIndex(int num)`

- **原型**：`monster * GetMonsterIndex(int num)`
- **形参**：
  - `num`：`int` — 怪物索引
- **返回值**：`monster *` — 成功返回怪物指针，索引越界返回 NULL
- **功能**：根据索引获取全局怪物数组中的怪物指针

- **功能**：游戏主循环，显示游戏菜单（查看属性、背包管理、商店购买、前往地图、保存退出），处理用户选择

## 九、数据结构定义汇总

### 核心结构体关系图

```
User (角色)
├── 基础属性: uname, level, exp, gold
├── 成长属性: strength, physique, agility
├── 战斗属性: max_hp, hp, max_mp, mp, attack, defense, speed
└── 背包: invertory_size, invertory[Item]
        └── Item: item_id, item_count

items (物品模板)
├── id, item, effect, price
└── 通过 itemList 管理

monster (怪物)
├── mon_name, lever, HP, Attack_power, Defense_power, speed
├── experience, gold
└── 通过 MapInfo.poll 关联到地图

MapInfo (地图配置)
├── ID, name, IsSecure
├── Min_level, Max_level
├── poll[5], rates[5], monster_num
└── 通过 Map_ID 索引
```

### 全局常量

| 常量名 | 值 | 定义位置 | 说明 |
|--------|-----|----------|------|
| `MAX_ITEMS` | 20 | save.h, item.h, save.c | 背包最大物品格数 |
| `MONSTER_MAX` | 50 | monster.h | 最大怪物数量 |
| `BUF_LEN` | 256 | monster.h | 读取缓冲区长度 |
| `MAP_HIGH` | 20 | map.h | 地图高度 |
| `MAP_WIDTH` | 70 | map.h | 地图宽度 |
| `DANGER_RATE` | 15 | map.h | 遇敌概率（%） |
