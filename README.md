# V1版本

##### 玩家数据 (save)
```
文件格式
角色名|等级|经验|金币
力量|体质|敏捷
气血上限|当前气血|法力上限|当前法力
攻击|防御|速度
背包(物品ID:数量,物品ID:数量,...)

一个文件对于一个用户
文件名通过创建角色名来创建文件，在save文件夹下
每次重新运行后，会读取save文件下的文件名，存放一个动态数组，
新游戏->创建角色名，先查看数组是否存在角色名，如果存在，提示角色名已创建

```


##### 升级逻辑

```
获得经验 → 累加至当前经验值
    ↓
计算当前等级所需升级经验
    ↓
[当前经验 >= 所需经验]?
    ↓[是]                ↓[否]
扣除升级经验            升级流程结束
等级+1
获得属性点
刷新角色属性
    ↓
[经验仍 >= 新等级所需经验]?
    ↓[是]                ↓[否]
循环升级                升级流程结束

怪物经验奖励 = 怪物等级 × 经验系数

```

​       阶段一：1-10级（新手期）

- 公式： exp = level × 100
- 示例：1级升2级需要100经验，10级升11级需要1000经验
- 特点：线性增长，升级非常快，让新手快速入门
阶段二：11-30级（成长期）

- 公式： exp = level² × 2 + level × 200
- 示例：15级升16级需要 15²×2 + 15×200 = 450 + 3000 = 3450 经验
- 特点：二次函数增长，升级速度开始明显变慢
阶段三：31-60级（成熟期）

- 公式： exp = level² × 5 + level × 400
- 示例：40级升41级需要 40²×5 + 40×400 = 8000 + 16000 = 24000 经验
- 特点：二次项系数加大，升级难度大幅提升
阶段四：60级+（高级期）

- 公式： exp = level² × 10 + level × 1000
- 示例：70级升71级需要 70²×10 + 70×1000 = 49000 + 70000 = 119000 经验
- 特点：二次项系数翻倍，后期升级非常困难
