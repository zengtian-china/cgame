# 文字版升级打怪游戏需求文档（精简版）

## 一、项目概述

### 1.1 项目名称
梦幻文字版 - 打怪升级

### 1.2 核心玩法
创建角色 → 探索地图 → 遇敌战斗 → 获得经验金币 → 升级提升属性 → 挑战更强怪物

### 1.3 技术栈
- **语言**：C语言（标准C99）
- **数据存储**：纯文本文件（.txt）
- **运行平台**：Windows控制台

## 二、角色系统

### 2.1 属性设定

| 属性 | 作用 |
|------|------|
| 等级 | 角色等级，决定属性上限 |
| 经验 | 当前经验值，达到阈值升级 |
| 气血 | 当前/最大生命值 |
| 法力 | 当前/最大魔法值 |
| 攻击 | 物理攻击力 |
| 防御 | 物理防御力 |
| 速度 | 决定战斗出手顺序 |
| 金币 | 游戏货币 |

### 2.2 属性成长

每升一级获得5点属性点，自由分配：
- 力量 → 攻击（1力量 = 0.8攻击）
- 体质 → 气血上限（1体质 = 5气血）
- 敏捷 → 速度（1敏捷 = 0.7速度）

防御 = 等级 × 2 + 装备加成

### 2.3 角色创建

- 输入角色名称（1-10字符）
- 选择初始加点方案：
  - 全力型：力量×5
  - 均衡型：力量×2 + 体质×2 + 敏捷×1
  - 敏捷型：敏捷×5

## 三、战斗系统

### 3.1 战斗规则

- **回合制**：双方轮流行动
- **速度判定**：速度高者优先出手
- **逃跑机制**：成功率 = 自身速度 / (自身速度 + 敌方速度)，失败则损失一回合
- **死亡处理**：损失当前经验的10%，返回城镇恢复满血

### 3.2 战斗操作

| 操作 | 效果 | 消耗 |
|------|------|------|
| 普通攻击 | 物理攻击敌方 | 0 |
| 技能攻击 | 使用技能造成伤害 | 法力 |
| 防御 | 本回合受到伤害减半 | 0 |
| 吃药 | 使用药品恢复气血/法力 | 消耗物品 |
| 逃跑 | 尝试逃离战斗 | 0 |

### 3.3 伤害公式

```
物理伤害 = (攻击 - 敌方防御) × 随机系数(0.8 ~ 1.2)
法术伤害 = 技能基础伤害 + 等级 × 2
```

### 3.4 技能系统

| 技能 | 消耗法力 | 效果 | 解锁等级 |
|------|----------|------|----------|
| 重击 | 10 | 造成1.5倍普通攻击伤害 | 1级 |
| 横扫千军 | 30 | 连续攻击敌方2次 | 10级 |
| 治愈术 | 20 | 恢复自身50+等级×5气血 | 5级 |

## 七、升级系统

### 7.1 经验获取

- 击杀怪物获得经验

### 7.2 升级经验

```
升级经验 = 当前等级 × 100
```

### 7.3 升级效果

- 等级+1
- 基础属性提升
- 获得5点属性点
- 解锁新技能（特定等级）

## 八、数据存储

### 8.1 文件结构

```
/game/
├── save/
│   └── player.txt       # 玩家存档
├── data/
│   ├── monster.txt      # 怪物数据
│   ├── item.txt         # 物品数据
│   └── map.txt          # 地图数据
```

### 8.2 数据格式

#### 玩家存档 (`player.txt`)
```
角色名|等级|经验|金币
力量|体质|敏捷
气血上限|当前气血|法力上限|当前法力
攻击|防御|速度
背包(物品ID:数量,物品ID:数量,...)
typedef struct user
{
    char uname[50]; //角色名
    int level;      //等级
    long exp;       //经验
    int gold;       //金币
    int strength;   //力量
    int physique;   //体质
    int agility;    //敏捷
    int max_hp;     //气血上限
    int hp;         //当前气血
    int max_mp;     //法力上限
    int mp;         //当前法力
    int attack;     //攻击
    int defense;    //防御
    int speed;      //速度
    Item invertory[MAX_ITEMS];
}User;
```

#### 怪物数据 (`monster.txt`)
```
怪物ID|名称|等级|气血|攻击|防御|速度|经验奖励|金币奖励
```

#### 物品数据 (`item.txt`)
```
物品ID|名称|类型|效果|效果值|价格|描述
```

#### 地图数据 (`map.txt`)
```
地图ID|名称|最低等级|最高等级|是否安全区|怪物列表(怪物ID:概率,...)
```

## 九、游戏流程

```
开始游戏
    ↓
[主菜单]
├── 新游戏 → 创建角色 → 进入游戏
└── 继续游戏 → 读取存档 → 进入游戏
    ↓
游戏主界面
    ↓
[菜单选项]
├── 查看属性
├── 背包管理
├── 升级加点 //
├── 商店购买
├── 前往地图
└── 保存退出
    ↓
前往地图 → 行走遇敌 → 战斗 → 获得奖励 → 继续探索
```
